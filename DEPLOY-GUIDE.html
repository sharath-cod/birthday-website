GitHub Pages (Free — Recommended)
1
Create a free GitHub account
Go to github.com → click "Sign up" → use any email → it's completely free.
Remember your username — you'll need it later.
💡 Already have GitHub? Skip to step 2.
2
Create a new repository
After logging in, click the green "New" button (top left) → Name it exactly: birthday-website → Set to Public → Click "Create repository"
github.com/new
Repository name: birthday-website · ● Public · ○ Private
3
Upload your website files
In your new repository → click "uploading an existing file" link → Unzip your birthday-website.zip → Drag ALL files from inside it into the browser window.
What to upload (contents of birthday-website/): 📄 index.html ← main page 📁 css/ 📄 style.css 📁 js/ 📄 app.js ← edit this for photos/videos 📁 images/ ← your photos go here 📁 videos/ ← your videos go here
⚠️ Upload the FILES inside the zip, not the zip file itself or the birthday-website folder.
4
Enable GitHub Pages
In your repository → click Settings (top tabs) → scroll to Pages (left sidebar) → Under "Source" select Deploy from a branch → Branch: main → folder: / (root) → click Save
⏱️ Wait 2–3 minutes. GitHub will give you a URL like: yourusername.github.io/birthday-website — it works immediately!
5
Connect your sharathshari.tech domain
Still in GitHub Pages settings → under "Custom domain" → type sharathshari.tech → click Save.

Now go to Hostinger hPanel → Domains → sharathshari.tech → DNS / Nameservers and add these records:
# Add these 5 DNS records in Hostinger: Type Name Value TTL A @ 185.199.108.153 3600 A @ 185.199.109.153 3600 A @ 185.199.110.153 3600 A @ 185.199.111.153 3600 CNAME www YOUR-GITHUB-USERNAME.github.io 3600
⏳ DNS changes take 10 min – 24 hours to fully work. Usually it's done in under an hour.
🔒 Back in GitHub Pages, tick "Enforce HTTPS" for a padlock on your site — free SSL!
6
🎉 Your site is live at sharathshari.tech!
Visit sharathshari.tech in your browser. The birthday website is now live for your girlfriend to see!

How to Add Your Photos & Videos
1
Put your files in the right folders
Photos → images/ folder  |  Videos → videos/ folder  |  Music → also in videos/
2
Open js/app.js in Notepad
Find the PHOTOS section and update the src values:
// BEFORE (placeholder): { src: '', label: 'Our first photo 💕', height: 260 }, // AFTER (your real photo): { src: 'images/photo1.jpg', label: 'Our first date 💕', height: 260 }, { src: 'images/holiday.jpg', label: 'Summer holiday 🌅', height: 220 },
3
Do the same for VIDEOS
// Local .mp4 file: { src: 'videos/clip.mp4', title: 'Our Holiday 🌊', desc: 'Best trip ever!', date: 'Aug 2024' }, // YouTube video (use embed URL): { youtube: 'https://www.youtube.com/embed/VIDEO_ID', title: 'Our Song 🎵', desc: '...', date: '...' },
4
Set her birthday date & name
const HER_NAME = "Priya"; ← her name const BIRTHDAY = new Date(2025, 5, 15); ← June 15, 2025 const MUSIC_SRC = "videos/our-song.mp3"; ← background music
📅 Months are 0-indexed: Jan=0, Feb=1, Mar=2, Apr=3, May=4, Jun=5, Jul=6, Aug=7, Sep=8, Oct=9, Nov=10, Dec=11
5
Re-upload the updated files
After editing app.js and adding photos/videos, re-upload to GitHub (drag & drop in browser) or to Hostinger File Manager. Changes go live in seconds!

Pro Tips for the Best Experience
📷
Compress photos before uploading
Use squoosh.app (free, browser-based) — compress to <300KB each. Site loads much faster. Format: WebP or JPEG.
🎬
Compress videos before uploading
Use Handbrake (free app) — compress to H.264 MP4, quality ~22. Keep under 50MB per video for fast loading.
🎵
Free romantic background music
Download free royalty-free music from pixabay.com/music — search "romantic piano" or "love song". Download as MP3, put in videos/ folder.
📱
Test on mobile before sending the link
Open the URL on your phone first to make sure everything looks perfect before she sees it!
